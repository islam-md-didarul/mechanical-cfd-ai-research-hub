# Upload and Publication Guide

## Recommended repository name

`mechanical-cfd-ai-research-hub`

## GitHub website method

1. Open https://github.com/new
2. Set owner to `islam-md-didarul`.
3. Enter repository name `mechanical-cfd-ai-research-hub`.
4. Use this description:

   `Curated research and learning hub for mechanical engineering, CFD, numerical methods, AI4CFD, reduced-order modeling, FEA, and scientific communication.`

5. Select **Public**.
6. Do not initialize with another README or license because this package already contains them.
7. Create the repository.
8. On the empty repository page, choose **uploading an existing file**.
9. Upload the contents of this folder, preserving its directory structure.
10. Commit with:

   `Create mechanical engineering CFD and AI research hub`

## Command-line method

```bash
git init
git add .
git commit -m "Create mechanical engineering CFD and AI research hub"
git branch -M main
git remote add origin https://github.com/islam-md-didarul/mechanical-cfd-ai-research-hub.git
git push -u origin main
```

## After upload

1. Add repository topics:
   - mechanical-engineering
   - computational-fluid-dynamics
   - ai4cfd
   - numerical-methods
   - reduced-order-modeling
   - dynamic-mode-decomposition
   - koopman-operator
   - pinns
   - finite-element-analysis
   - scientific-machine-learning
2. Pin the repository on your GitHub profile.
3. Open **Actions** and confirm that the link-check workflow passes.
4. Add the hub link to your profile README.

# Python and Mathematics Foundations

## Objective

Build the programming and mathematical background required for numerical CFD, machine learning, DMD, Koopman methods, and PINNs.

## Recommended order

1. [ehmatthes/pcc_3e](https://github.com/ehmatthes/pcc_3e)
2. [krishnaik06/Complete-Python-Bootcamp](https://github.com/krishnaik06/Complete-Python-Bootcamp)
3. [HenryNdubuaku/maths-cs-ai-compendium](https://github.com/HenryNdubuaku/maths-cs-ai-compendium)
4. [jonkrohn/ML-foundations](https://github.com/jonkrohn/ML-foundations)
5. [shaiful019/MLOB1](https://github.com/shaiful019/MLOB1)

## Expected outcomes

- Write reliable Python scripts and notebooks.
- Manipulate arrays, files, tables, and scientific data.
- Understand vectors, matrices, eigenvalues, SVD, derivatives, probability, and statistics.
- Prepare datasets and evaluate machine-learning models.

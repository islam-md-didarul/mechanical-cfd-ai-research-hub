# Scientific Communication

## Writing

- Use [islam-md-didarul/LaTex-Thesis-Template](https://github.com/islam-md-didarul/LaTex-Thesis-Template) as a structural reference.
- Follow the target journal, conference, or university template rather than assuming one format fits all.
- Track software versions, datasets, boundary conditions, and validation evidence.

## Presentation and rehearsal

- Use [islam-md-didarul/voice-pro](https://github.com/islam-md-didarul/voice-pro) optionally for transcription and rehearsal.
- Prepare one research story: problem → gap → method → validation → findings → implications.
- Keep claims traceable to figures, data, equations, and references.

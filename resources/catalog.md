## Classification

| Label | Meaning |
|---|---|
| **Core** | Directly supports the main CFD–AI research pathway |
| **Supporting** | Strengthens important prerequisite or implementation skills |
| **Specialized** | Useful for a specific method or application |
| **Reference** | Mainly used for discovering external resources |
| **Optional** | Helpful for productivity or communication |

## Adding another resource

1. Verify the original repository and license.
2. Review its maintenance status and technical relevance.
3. Assign a category, level, and classification.
4. Add it to this catalog and `catalog.yml`.
5. Connect it to at least one learning path or project guide.
6. Run the link-check workflow.
